# Test Julia

This is a Next.js application with TypeScript and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Next.js 14
- TypeScript
- Tailwind CSS
- ESLint
- App Router
